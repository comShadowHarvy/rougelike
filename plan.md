# Roguelike Game Project Plan

## Core Features

- [x] CLI-based game
- [x] Class-based characters
- [x] Procedurally generated dungeons
- [x] Turn-based combat
- [x] Player movement
- [x] Monster generation and AI
- [x] Leveling system
- [x] Permadeath

## Advanced Features

- [x] Stairs and multiple levels
- [x] Field of View (FOV)
- [x] Corpses
- [x] More colors
- [x] Boss rooms and bosses
- [x] Legendary chests and items
- [x] Dynamic map size
- [x] Auto-summoning for certain classes
- [x] Chests and loot system
- [x] Inventory and equipment system
- [x] Special abilities for all classes
- [x] Auto-combat uses special abilities
- [x] Traps and Hazards

## Gameplay Features

- [ ] Hunger/food system
- [ ] Status effects (poisoned, stunned, burning, frozen, etc.)
- [ ] Terrain features (water, lava, chasms, doors, altars)
- [ ] Crafting/alchemy system
- [ ] Quests/events system
- [ ] NPCs/merchants
- [ ] Pets/familiars
- [ ] Achievements
- [ ] Multiple dungeon biomes/variety

## UI/UX

- [ ] Replace console.log with TUI library (blessed/ink)
- [ ] Mouse support
- [ ] Health bars and mini-map
- [ ] Colored message log with filtering
- [ ] Settings/config file

## Quality of Life

- [ ] Multiple save slots
- [ ] Auto-explore feature
- [ ] Macros for ability combos
- [ ] Better death screen with stats

## Controls

- **Movement:** `w`, `a`, `s`, `d`
- **Heal:** `h`
- **Summon:** `u`
- **Use Ability:** `b`
- **Toggle Auto-Combat:** `c`
- **Toggle Auto-Heal:** `x`
- **Equip Item:** `e`
- **Go down stairs:** `>`
- **Go up stairs:** `<`
- **Exit:** `ctrl+c`
